CSS Classes:

  • .company-stats-responsive - Responsive component (Mobile → Desktop)
  • .company-stats-xl - Desktop XL fixed size (368px)
  • .company-stats-sm - Mobile SM fixed size (328px)
  • .company-stats-grid-auto - Auto responsive grid (1→2→3 columns)
  • .company-stats-figma-grid - Exact Figma grid layout
  • .company-stats-variant-[primary|secondary|dark] - Color variants

Technical Specifications

Mobile SM (Figma: 1818:8969)

  • Dimensions: 328px width × variable height
  • Border Radius: 0px 0px 36px 0px
  • Border: 1px right + bottom (#E7E5E4)
  • Padding: 0px 16px 16px 0px
  • Gap: 14px (main), 8px (text group)
  • Number: 26px, line-height 1.23, letter-spacing 1.54%
  • Label: 16px, line-height 1.5
  • Subline: 16px, line-height 1.625
