Image Slider Component

Responsive Image Slider Komponente mit Navigation Buttons und Pagination Dots. Automatische Anpassung zwischen Mobile (Touch Navigation, 185px) und Desktop (Button Navigation, 434px). Vollständig mit Tailwind Utility Classes implementiert.
Desktop: Node 1780-35047 (1064px Container, 768×434px Images) | Mobile: Node 1731-23716 (Full Width, 185px Height)
