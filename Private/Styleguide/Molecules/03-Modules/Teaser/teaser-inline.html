CSS System

Responsive Class System

Mobile-First Approach

Pixelgenaue Figma-Umsetzung mit Responsive Breakpoints

Mobile First (bis 1023px)
/* Mobile: 328px vertikal - Figma SM */
.teaser-inline-responsive {
  flex-direction: column;
  max-width: 328px;
  border-radius: 36px 0px 36px 0px;
}

.teaser-inline-responsive-image {
  width: 100%;
  height: 185px;
  border-radius: 36px 0px 0px 0px;
}

.teaser-inline-responsive-content {
  gap: 20px;
  padding: 24px;
}
Desktop Breakpoint (1024px+)
/* Desktop: 768px horizontal - Figma XL */
@media (min-width: 1024px) {
  .teaser-inline-responsive {
    flex-direction: row;
    max-width: 768px;
    border-radius: 64px 0px 64px 0px;
  }
  
  .teaser-inline-responsive-image {
    width: 268px;
    height: 268px;
    border-radius: 64px 0px 0px 0px;
  }
  
  .teaser-inline-responsive-content {
    gap: 24px;
    padding: 32px;
  }
}
Typography (Figma-Spezifikationen)
/* Mobile Typography */
.teaser-inline-responsive-overline {
  font: 600 12px/1.17 "Titillium Web";
  letter-spacing: 13.33%;
}

.teaser-inline-responsive-headline {
  font: 400 20px/1.4 "Titillium Web";
}

/* Desktop Typography */
@media (min-width: 1024px) {
  .teaser-inline-responsive-headline {
    font-size: 24px;
    line-height: 1.33;
  }
}
